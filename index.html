Índice NDMI / Moisture Index

Visor de humedad de la vegetación obtenido a partir de imagen Sentinel-2. Valores altos indican mayor contenido de humedad; valores bajos indican menor humedad o posible estrés hídrico.

Control de visualización

Puedes cambiar el mapa base, activar o desactivar la capa NDMI y ajustar la opacidad para ver mejor el relieve, la ortofoto o el mapa base.

Fuente: Copernicus Sentinel-2 L2A. Capa visual exportada como KMZ.
